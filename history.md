# Fact 1

Git is Created in 2008 by Tom Preston-Werner, Chris Wanstrath, PJ Hyett and Scott Chacon, GitHub is a web-based online source code host that uses Git (a distributed version control system developed by Linus Torvalds) to manage version control of applications. The aim was to help developers more easily collaborate on their code through the use of tools that allow for the sharing of code, the tracking of changes to code, and the management of projects via the internet.
